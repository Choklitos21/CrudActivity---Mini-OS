DESCRIPCION GENERAL DEL PROYECTO
Chokintosh OS es un proyecto académico que consiste en la creación de un sistema operativo simulado desde cero utilizando HTML, CSS y JavaScript.
Está fuertemente inspirado en el diseño original de macOS Tahoe, tomando la estética, la distribución visual y varias aplicaciones icónicas del sistema de Apple.
Este proyecto busca recrear de manera fiel la experiencia visual y parcial de uso de un sistema operativo moderno. Aunque aún se encuentra en fase de desarrollo, presentamos una versión conceptual y funcional de lo que podría ser la evolución del sistema al cual rendimos homenaje.
Chokintosh OS pretende mostrar:
Un diseño minimalista y moderno.
Un entorno visual completo y organizado.
Una colección de aplicaciones base simuladas.
Una demostración de navegación básica dentro de un “sistema operativo” web.
El objetivo principal es mostrar una vista previa de un posible nuevo sistema operativo que reemplaza la experiencia actual de macOS, manteniendo la esencia original, pero con una visión propia del equipo de desarrollo.

OBJETIVO DEL PROYECTO
    -Nuestro propósito fue presentar un boceto funcional que muestre:
    -Interfaz gráfica del sistema operativo.
    -Comportamiento de ventanas, iconos y aplicaciones.
    -Animaciones y estilo característico de macOS.
    -Algunas funciones básicas simuladas dentro de cada app.
Aunque el proyecto aún no incluye toda la funcionalidad real de un sistema operativo, sí ofrece una experiencia visual sólida, coherente y en constante desarrollo.

APLICACIONES INCLUIDAS EN CHIKINTOSH OS
A continuación explicamos cada una de las apps creadas, su propósito dentro del proyecto y qué representan dentro del entorno del sistema.

1. APPs — Panel principal de aplicaciones
Esta aplicación funciona como el launcher o “App Library” del sistema operativo.
¿Qué muestra?
Íconos de todas las aplicaciones instaladas.
Una interfaz muy similar al navegador de apps de macOS.
Organización visual clara, minimalista y responsiva.
¿Qué representa?
La verdadera biblioteca de aplicaciones de macOS.
La entrada principal para acceder a todo el entorno del sistema.

2. Safari — Navegador web simulado
Safari es una copia visual del navegador original de macOS.
Características:
Interfaz completamente recreada.
Barra de direcciones, botones, y diseño idéntico al del sistema original.
Animaciones y estética fiel al navegador real.
¿Qué representa?
El navegador oficial del sistema operativo.
En futuras versiones se espera conectarlo realmente con internet para hacer búsquedas y cargar páginas.

3. Música — Reproductor multimedia simulado
La aplicación Música replica la estética del reproductor multimedia de macOS.
Funciones y características:
Interfaz moderna inspirada en la app Música nativa de Apple.
Diseño idéntico en colores, tipografías y distribución.
Lista de canciones simulada (solo presentación visual por ahora).
¿Qué representa?
El sistema de música del OS, mostrando cómo luciría un reproductor completo dentro de Chokintosh OS.
Una copia muy fiel del diseño original.

4. Terminal — Emulador de línea de comandos
La app Terminal es una recreación exacta de la terminal de macOS.
Funcionalidad:
Permite escribir texto como si fuera una consola real.
Estética idéntica: colores, tipografías, diseño de ventana, barra de herramientas.
Soporta ingreso de comandos a nivel visual (no ejecuta lógica avanzada por ahora).
¿Qué representa?
Una representación del entorno Unix de macOS.
La “ventana del sistema” donde los usuarios avanzados interactúan con comandos.

5. Finder — Navegador de archivos
Finder emula el clásico explorador de archivos del ecosistema Apple.
Características:
Grilla de carpetas y archivos simulados.
Vista elegante, ordenada y fiel al diseño de macOS.
Ventanas con barra lateral, íconos y estructura realista.
¿Qué representa?
El sistema de navegación por carpetas del OS.
La base para construir un administrador de archivos real en el futuro.

ESTADO ACTUAL DEL PROYECTO
Actualmente, Chokintosh OS es una versión prototipo del sistema final.
No todas las funciones están completadas, pero la base visual y conceptual está lista.

Estado actual:
- Interfaz del sistema operativo
- Ventanas y diseño general
- Apps simuladas y funcionalidad básica
- Funciones reales de sistema (navegación real, internet, archivos) en desarrollo
- Integración completa entre aplicaciones pendiente
- Proyecto en construcción

MISION Y VISION DEL PROYECTO

Misión
Crear una experiencia visual completa, inmersiva y moderna que permita simular un sistema operativo desde el navegador.

Visión
Construir un sistema operativo web que reemplace y mejore la experiencia actual de macOS, introduciendo nuevas funciones, diseño renovado y fluidez en cada aplicación.

CONCLUSION
Chokintosh OS es un proyecto que combina diseño, desarrollo web y creatividad.
Representa un paso hacia un sistema operativo visual, moderno e inspirado en macOS, pero con identidad propia. Aunque aún está en crecimiento, esta presentación demuestra el potencial del equipo y del concepto general del proyecto.
